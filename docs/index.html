DPAF-Net: Dual-Pattern Learning for Fire–Smoke Instance Segmentation
with a Large-Scale Dataset

Muhammad Ayaz1, Sareer ul Amin1, Hikmat Yar2, Salman Khan3, Wonseop Shin1, Sanghyun Seo1,*
1 Chung-Ang University, Republic of Korea  |  2 KAIST, Republic of Korea  |  3 Oxford Brookes University, United Kingdom
* Corresponding Author  ·  sanghyun@cau.ac.kr
Paper (Soon) Code (Soon) Dataset (Soon)
tl;dr  DPAF-Net explicitly separates fire (high-freq 3×3 kernels) and smoke (low-freq 5×5/7×7 kernels) via a Dual-Pattern Encoder, fuses them with pixel-wise softmax attention, and achieves 40.2% AP at 17.2 FPS on a new 7,917-image dual-class dataset — the largest instance-level fire & smoke benchmark to date.

Abstract

Uncontrolled fire and smoke events cause thousands of fatalities and billions in infrastructure damage annually, yet existing automated detection systems still lack the precision needed for life-critical response. Existing segmentation methods rely on shared feature representations that conflate fire's high-frequency textures and smoke's diffuse, semi-transparent structures, while semantic approaches lack instance-level discrimination and datasets remain limited to single-class annotations.

We introduce DPAF-Net, a novel instance segmentation framework built on three components: a Dual-Pattern Encoder (DPE) with fire-specific 3×3 kernels and smoke-specific large-kernel convolutions (5×5, 7×7) for complementary feature extraction; an Adaptive Fusion Module (AFM) with pixel-wise softmax attention to dynamically weight fire and smoke features per spatial location; and a DCNv2-enhanced ResNet-50 backbone for adaptive modeling of irregular boundaries.

DPAF-Net achieves 40.2% AP at 17.2 FPS, outperforming the best baseline by +5.2% AP, with strong boundary precision (AP75: 40.0%), small-object detection (APS: 30.4%), and a mean AP of 45.1%. We further release a large-scale dataset of 7,917 images with dual-class instance-level annotations across diverse real-world environments.

The Dual-Pattern Encoder (DPE) processes FPN features through two parallel branches: a fire branch employing 3×3 convolutions to capture high-frequency textures and color gradients, and a smoke branch using large-kernel convolutions (5×5, 7×7) to model low-frequency, dispersed, semi-transparent patterns. Both branches include Pattern-Specific Channel Attention (PSCA) for per-branch recalibration (256→16→256, r=16).

The Adaptive Fusion Module (AFM) applies a 3-stage spatial attention mechanism that generates pixel-wise fusion weights via softmax normalization, enabling the network to adaptively emphasize fire or smoke features at each spatial location — particularly effective in scenes where both classes coexist or overlap.
